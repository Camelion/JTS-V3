Безымянный Дух:
Тень Халиша находится в последней комнате каждой гробницы. В каждой гробнице, а их четыре, есть своя тень. У каждой из теней есть четыре кубка. Между прочим, четыре тени Халиша изменяют свое местоположение всякий раз, когда они появляются.Это может оказаться тяжелее, чем Вы думали!
