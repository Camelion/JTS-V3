=====Способ призыва стихийных предметов=====
//summon_attribute [item_id] [type(0~5)] [value] [enchant][option normal][option random]

Item_id: ID предмета
type: Тип стихии (0:Огонь, 1:Вода, 2:Ветер, 3:Земля, 4:Свет, 5:Тьма)
value: Параметр стихии
Enchant: Уровень зачарования
[option normal][option random] Номер опции улучшения

Пример)
//summon_attribute 9416 1 200 6 0 0
(Кираса Венеры Стихия Воды: 200 Зачарование: 6 Улучшений нет)
