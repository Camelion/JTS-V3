Арбалетчик - класс, который обладает навыками владения арбалетом и использования ловушек, в него входят исключительно женщины Камаэль.
Вы можете также приобрести другие навыки, полезные для выживания. Этот класс больше подходит для исследования или шпионажа, чем для того, чтобы биться с врагом один на один.
