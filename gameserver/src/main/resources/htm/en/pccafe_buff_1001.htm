Adventurer Guildsman:
Adventurers below level 40 can receive the following Supplemental Magic.
The warrior buff set, involves Berserker Spirit, Might, Shield, Wind Walk, Focus, Death Whisper, Guidance, Bless Shield, Blessed Soul, Haste, Agility, and Vampiric Rage.The wizard buff set, has Berserker Spirit, Shield, Wind Walk, Blessed Body, Blessed Soul, Magic Barrier, Empower, and Acumen.
Warrior buff set - 3100 points
Wizard buff set - 1600 points
Select individual buffs to receive.
