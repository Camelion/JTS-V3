=====How to summon item attribute=====
//summon_attribute [item_id] [type(0~5)] [value] [enchant][option normal][option random]

Item_id : Item ID
type : Attribute Type (0:Fire, 1:Water, 2:Wind, 3:Earth, 4:Divinity, 5:Darkness)
value : Attribute Value
Enchant : Enchant Level
[option normal][option random] Augmenting Option Number

ex)
//summon_attribute 9416 1 200 6 0 0
(Vesper heavy armor Water attribute:200 Enchant:6 No augmenting)
